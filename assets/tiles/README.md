# Mahjong Tile SVGs

This directory contains SVG images for all 144 mahjong tiles.

## Required Files

### Suit Tiles (108 total)
Bamboo (索): `bamboo_1.svg` through `bamboo_9.svg`
Character (萬): `character_1.svg` through `character_9.svg`
Dot (筒): `dot_1.svg` through `dot_9.svg`

### Honor Tiles (28 total)
Winds: `wind_east.svg`, `wind_south.svg`, `wind_west.svg`, `wind_north.svg`
Dragons: `dragon_red.svg`, `dragon_green.svg`, `dragon_white.svg`

### Bonus Tiles (8 total)
Flowers: `flower_1.svg` through `flower_4.svg`
Seasons: `season_1.svg` through `season_4.svg`

## Naming Convention

- Suit tiles: `{suit}_{number}.svg` (e.g., `bamboo_1.svg`)
- Wind tiles: `wind_{direction}.svg` (e.g., `wind_east.svg`)
- Dragon tiles: `dragon_{color}.svg` (e.g., `dragon_red.svg`)
- Flower tiles: `flower_{index}.svg` (1-4)
- Season tiles: `season_{index}.svg` (1-4)

## Design Guidelines

- Size: 60x80 pixels (3:4 aspect ratio)
- Style: Clean, modern with traditional elements
- Colors: Match the 70s retro palette
  - Bamboo: Avocado green (#568203)
  - Character: Burnt orange (#CC5500)
  - Dot: Complement blue (#4169E1)
  - Honor: Purple (#6B3FA0)
  - Bonus: Gold (#FFD700)
